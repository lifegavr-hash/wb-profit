**Moneyplace прекратил работу в ноябре 2025.** Это был один из 3-4 крупных игроков с фокусом на финансовую аналитику и юнит-экономику (3 990 ₽/мес, AI-описания, обновление 4 раза/день). **Их аудитория сейчас в поиске замены** — для нас это окно возможности.

## КАТЕГОРИЯ 1: Прямые конкуренты (фокус на финансовой аналитике/прибыли)

Это **наша ниша**. Селлер заходит, чтобы понять «сколько я реально заработал».

### MPFinAssist
- **URL:** https://mpfinassist.io/
- **Фокус:** Финансовая аналитика + учёт для WB, Ozon, Я.Маркет. P&L, ДДС, журнал транзакций.
- **Цена:** уточнять на сайте (не выложена публично — обычно ~2000-5000 ₽/мес).
- **Плюсы:** мульти-маркетплейс, отчёты по управленческому учёту.
- **Минусы:** интерфейс «бухгалтерский» — пугает обычного селлера.
- **Сравнение с нами:** более бухгалтерский, мы можем выделиться простотой.

### SoykaSoft
- **URL:** https://soykasoft.ru/features/finansovyj-uchyot-dlya-marketplejsa/finansovyj-uchet-wildberries/
- **Фокус:** Финансовый учёт WB через API — продажи, возвраты, комиссии, штрафы, расходы, выплаты. Реальная прибыль с учётом всех комиссий, логистики, возвратов, налогов.
- **Цена:** на сайте.
- **Сравнение с нами:** **прямой клон по концепции**. Изучить, чем мы отличаемся.

### Litestat
- **URL:** https://litestat.io/wildberries/
- **Фокус:** Управленческий учёт, финансовая аналитика, прогноз прибыли, контроль расходов.
- **Цена:** на сайте.
- **Сравнение с нами:** также конкурент в нашей нише.

### Fin-Ctrl (Финансист)
- **URL:** https://fin-ctrl.ru/wildberries
- **Фокус:** Юнит-экономика + управление финансами, **интеграция с 1С**.
- **Цена:** на сайте.
- **Сравнение с нами:** 1С — это у нас слабое место. Если селлер хочет 1С — пойдёт к ним.

### Sellerdata.ru
- **URL:** https://sellerdata.ru/
- **Фокус:** Бесплатная финансовая аналитика для WB. Калькулятор прибыли + отчёты по продажам.
- **Цена:** бесплатно (вопрос — как монетизируются).
- **Сравнение с нами:** опасный — бесплатно. Селлер может пойти туда, если у нас будет даже малая цена. Проверить, что они скрывают/какие лимиты.

### MoneyPlace [УМЕР]
- **URL:** https://moneyplace.io/ (часть страниц ещё доступна, но сервис закрыт)
- **Был:** Юнит-экономика, AI-описания, обновление 4 раза/день, 3 990 ₽/мес.
- **Статус (с ноября 2025):** прекратил работу.
- **Возможность:** их пользователи сейчас ищут замену. **Это твоя главная цель** в первой волне раздачи промокодов.

## КАТЕГОРИЯ 2: Универсальные гиганты (большие, дорогие, всеобъемлющие)

Это игроки с долей рынка. Селлер часто использует их по умолчанию.

### MPStats
- **URL:** https://mpstats.io/
- **Фокус:** Внутренняя + внешняя аналитика. SEO, рекламные кампании, отслеживание конкурентов, финансовый учёт. Топ-1 на рынке.
- **Цена:** базовый — **4 800 ₽/мес**, полный «All MPStats» — **29 400 ₽/мес**.
- **Аудитория:** топ-1000 селлеров по обороту.
- **Сравнение с нами:** мы дешевле и проще. Они «всё в одном» за дорого — мы «именно прибыль, без лишнего» за разумные деньги.

### Маяк (Mayak)
- **URL:** через поиск, нет единой посадочной (Chrome-расширение + сайт)
- **Фокус:** базовая аналитика, начинающие селлеры.
- **Цена:** есть бесплатный план + платные от ~1000-6000 ₽/мес.
- **Аудитория:** **500 000+ селлеров** — самая массовая база.
- **Сравнение с нами:** Маяк — это «для новичков, бесплатно». Мы — «для тех, кто перерос Маяк, но не дорос до MPStats». Между ними дыра — там наша ниша.

### EGGHEADS
- **URL:** https://chrome-stats.com/d/eabmbhjdihhkdkkmadkeoggelbafdcdd
- **Фокус:** Внутренняя + внешняя аналитика, удобный интерфейс, для новичков и опытных.
- **Цена:** от 6 000 ₽/мес.
- **Статус:** **официальный партнёр WB**.
- **Сравнение с нами:** партнёрство с WB — их сильное оружие. Нам это в перспективе тоже надо.

### MarketGuru
- **URL:** https://marketguru.io/
- **Фокус:** Бесплатное расширение для базовой аналитики WB.
- **Цена:** бесплатно + платный.
- **Сравнение с нами:** простой массовый продукт. Не прямой конкурент по прибыли.

### Stat4Market
- **URL:** https://stat4market.com/
- **Фокус:** Аналитика продаж и рекламы, быстрый старт через WB API, юнит-экономика.
- **Цена:** на сайте.
- **Сравнение с нами:** широкий профиль, мы — сфокусированный.

### SellerStat
- **URL:** через поиск.
- **Фокус:** WB + Ozon + Я.Маркет + МегаМаркет. Отчёты по товарам, категориям, селлерам, брендам. API-кабинеты, реклама.
- **Цена:** на сайте.
- **Сравнение с нами:** многомаркетплейсный, у нас только WB пока. Преимущество для тех, у кого 2-3 маркетплейса.

**Наш калькулятор работает только если он:** интегрирован с реальными данными юзера (т.е. подтягивает себестоимости из истории, реальные комиссии, реальную логистику) — это даёт результат который не получишь в калькуляторе на сайте.

## АНАЛИЗ: где наша позиция

### Прямые конкуренты (нас будут сравнивать с ними)
1. **MPFinAssist** — финансовая аналитика, мульти-маркетплейс
2. **SoykaSoft** — почти точный клон концепции
3. **Litestat** — управленческий учёт
4. **Fin-Ctrl** — финансы + 1С
5. **Sellerdata** — бесплатно, опасно для нас

### Наши потенциальные преимущества
- **Сфокусированность** — мы только про прибыль, не пытаемся быть «всё в одном» как MPStats.
- **Простота** — селлеру не нужно учиться 2 недели как в MPStats.
- **Цена** — должна быть в коридоре 500-2000 ₽/мес (ниже MPStats, сопоставима с Маяком). Сейчас твои тарифы 490/990/1990 ₽ — попадаешь в зону, **где наименее конкурентно**.
- **2 кабинета WB на Бизнесе** — стандартная фича, не выделяет.
- **Калькулятор юнит-экономики** — НЕ дифференциатор сам по себе, **но** с подтягиванием реальных данных юзера — это уже сильнее.

### Наши слабые места vs гиганты
- **Нет статуса «сертифицирован WB»** — у Анабар, Эггhеads, Sellematics есть. Это даёт доверие.
- **Нет анализа конкурентов / поиска ниш / SEO** — селлер часто хочет всё в одном месте.
- **Нет репрайсера / автоматизации цен** — серьёзная фича большинства конкурентов.
- **Нет интеграции с 1С** — отсекает крупных селлеров с бухгалтером.
- **Нет API для интеграций** — у нас читающий, у MPStats есть отдающий API.
- **Только WB** — без Ozon и Я.Маркет селлер с 2 маркетплейсами не возьмёт.

### Стратегические идеи (для разговора)
1. **Целиться в осиротевших юзеров Moneyplace** — они потеряли любимый сервис в ноябре 2025, ищут замену. Это **точечная аудитория**, которой можно дать промокод первой.
2. **Не пытаться конкурировать со всеми** — мы не MPStats. Мы — «сервис ясной прибыли для селлера 0.5-5 млн ₽/мес оборота, у которого нет времени и денег на MPStats».
3. **Получить статус партнёра WB** — в долгую важнее любых фич.
4. **Ozon как второй маркетплейс** — на горизонте 6-12 месяцев, как только бета WB стабилизируется.
5. **Не конкурировать в калькуляторе** — там 10+ бесплатных. Калькулятор как часть **общей экосистемы данных юзера** — это уже наше.
