# Первое письмо бета-тестеру с промокодом

Шаблон для первого контакта с человеком, которому ты раздаёшь промокод. Подойдёт для отправки в Telegram-личку, на почту или через форму обратной связи.

```

аналитика чистой прибыли для селлеров Wildberries.

🎟️ Промокод: XXXX-XXXX (вводится при регистрации)
🔗 Ссылка: https://[твой-домен].ru/login

1. Подключи WB API-токен в Настройках → увидишь свои реальные финансы за 30 дней
2. Открой Калькулятор → загрузи свои товары → найди убыточные позиции
3. Посмотри Аналитику → там история проблем по дням

Если что-то сломалось или непонятно — нажми кнопку «Сообщить
об ошибке» прямо в кабинете (она наверху на жёлтой полоске).
Я читаю каждое сообщение лично.

```

## Что подставить перед отправкой

- `[НОВОЕ_НАЗВАНИЕ]` — финальное имя проекта (решим завтра)
- `XXXX-XXXX` — конкретный промокод (генерируется через админку)
- `[твой-домен].ru` — новый домен после регистрации
- `[Твоё имя]` — как тебя зовут
- `[твой Telegram]` — `@username` для прямого связи

---

## Когда отправлять

- **Не массово** на старте — каждое письмо персонально.
- Сначала **20-30 человек**, отбор по: активный селлер WB (от 100к ₽ оборота/мес), готов писать фидбек, есть Telegram для быстрой связи.
- После первой недели и фиксов — следующие 50-100.

## Чего НЕ делать в первом письме

- Не давить с просьбой «расскажи всем друзьям» — отпугнёт. Пусть сначала сам попробует.
- Не писать «дайте отзыв на Trustpilot» — ещё рано.
- Не предлагать оплатить подписку прямо в первом письме — Trial и так бесплатный 14 дней.
