# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [v0.7.7.35] — 2026-05-28 (Pre-beta critical fixes)

### Исправлено (критические находки аудита)
- **Подделка тарифа из браузера**: убран прямой `sb.from('profiles').update({plan:'free'})`
  с фронта (`dashboard.html`). Истёкший тариф backend сам считает «expired» по
  `plan_expires_at` через `plan-check.js` — фронт больше не пишет в БД. Закрыта
  потенциальная дыра: юзер не может через DevTools присвоить себе бизнес-тариф.
- **Утечка данных между юзерами на одном ПК**: `logout()` теперь очищает все
  `wb_*` ключи в localStorage (+ `onboarding_done`, `calc_presets`, `col_prefs`,
  `calc_group_collapsed`), а не только `wb_token`. Второй селлер на том же ПК
  не увидит данные первого.
- **Онбординг новичка**: `onbFinish()` теперь создаёт запись в `wb_accounts`
  через `/api/profile?resource=wb-accounts` перед запуском `goFromHome()`.
  Раньше токен оставался только в `localStorage`, и при первом запросе
  бэк падал с 412 `NO_WB_ACCOUNT` — белый экран после онбординга.

### Изменено (false claims с лендинга)
- **«230+ селлеров уже считают»** убрано из hero (`index.html:851`) — продукт
  в инвайт-бете, цифра была недостоверной.
- **«AI советник»** удалён из feat-card и ai-section. Заменён на честные
  «История проблем» и «Калькулятор юнит-экономики» — фичи, которые реально
  работают в коде. CSS-метка `.ai-example::before` — «Подсказка» вместо
  «AI советник».
- **«Данные за 12 месяцев»** переписано: «Старт — 30 дней, PRO — 12 месяцев,
  Бизнес — 24 месяца» (в соответствии с тарифами).
- **«Данные в зашифрованном виде»** переписано: «HTTPS (TLS 1.3), RLS Postgres».
  Без обещания шифрования на уровне колонок (которого нет).
- **Тариф Бизнес**: описание актуализировано — «2 кабинета WB» убрано из
  «Скоро», т.к. фича работает с v0.7.7.30. Удалён «AI-чат с магазином (скоро)».

### Изменено (UX trial/expired баннеров)
- Кнопки «Оплатить →» и «Возобновить →» в trial/expired баннерах убраны —
  пока ЮKassa не подключена, они вели в Настройки на `alert('Платежи через
  ЮKassa подключим в ближайшем релизе')`. Тексты переписаны: «На период беты —
  бесплатно для всех приглашённых» / «Чтобы продлить — напишите через
  Сообщить об ошибке».

### Исправлено (мелочи)
- `api/config.js` health-check теперь отдаёт актуальную версию `v0.7.7.35`
  вместо захардкоженной `v0.7.7.19`.

## [v0.7.7.33] — 2026-05-28 (Бета-инфраструктура)

### Добавлено
- **BETA-полоска** в шапке dashboard.html — жёлтая фиксированная полоса
  «БЕТА-версия — данные могут быть неточными. Нашли ошибку? Напишите нам».
  Ссылка открывает модалку фидбэка. CSS-переменная `--beta-h:26px`
  сдвигает sidebar, topbar и mob-menu-btn в desktop и mobile media-блоках.
- **Модалка «Сообщить об ошибке»** в Настройках (категории
  bug/feature/question/praise/other) — `sb.rpc('submit_feedback', ...)`.
  При успехе — toast «Спасибо за обратную связь». Та же модалка из
  BETA-полоски.
- **Глобальный лог JS-ошибок** — `window.error` и `unhandledrejection` →
  `sb.rpc('log_frontend_error', ...)`. Установка в `init()` после
  `sb = createClient(...)`, защита от двойной инсталляции флагом
  `window.__errLoggerInstalled`.
- **Админ-раздел расширен 4 view**: `admin_beta_stats`,
  `admin_new_feedback`, `admin_available_codes`, `admin_beta_users`.
  Универсальный рендер таблиц из произвольных view (заголовки — ключи
  первой строки, ISO-даты конвертируются в ru-локаль).

### Изменено
- **Кнопка «📥 Скачать в Excel»** перенесена из спрятанного блока
