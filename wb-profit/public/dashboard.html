Добро пожаловать в WB Profit!
Это сервис, который покажет реальную прибыль по каждому товару на Wildberries, найдёт убыточные позиции и подскажет, где утекают деньги.
Чтобы начать работу, нам нужно подключиться к вашему кабинету WB через API токен с правами на чтение. Сервису нужны разделы «Статистика» (продажи, остатки) и «Продвижение» (рекламные расходы для точной прибыли).
🔑
Как получить API токен WB
  1. Откройте кабинет seller.wildberries.ru
  2. В правом верхнем углу — иконка профиля → Настройки
  3. Раздел Доступ к API
  4. Кнопка «Создать новый токен»
  5. Название: «WB Profit» · Тип: Только чтение
  6. Отметьте галочками разделы:
    • Статистика — продажи, заказы, остатки, отчёты
    • Продвижение — реклама (для точного расчёта прибыли)
    • Аналитика — финансовая отчётность
  7. Создать → скопировать токен
⚠️ Не делитесь токеном с третьими лицами. Если что — его можно отозвать в том же разделе WB.
🚀
Вставьте токен и запускаем
Вставьте скопированный токен сюда — мы сохраним его и сразу загрузим ваши данные.
Токен хранится только в вашем браузере и нигде не передаётся, кроме WB.

Подключение к Wildberries

Токен сохранён

Где взять токен: личный кабинет WB → Настройки → Доступ к API → Создать новый токен. Нужны разделы: Статистика, Продвижение, Аналитика.

Контроль кассового потока (необязательно)

Укажите ваши обязательные расходы в месяц (аренда, зарплата, налоги, кредиты, личные траты). Мы будем предупреждать, если прибыль от WB не покрывает эти расходы — это часто бывает причиной кассовых разрывов.
