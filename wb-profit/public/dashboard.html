Добро пожаловать в WB Profit!
Это сервис, который покажет реальную прибыль по каждому товару на Wildberries, найдёт убыточные позиции и подскажет, где утекают деньги.
Чтобы начать работу, нам нужно подключиться к вашему кабинету WB через API токен. Это безопасно — токен даёт только право читать статистику.
🔑
Как получить API токен WB
  1. Откройте кабинет seller.wildberries.ru
  2. В правом верхнем углу — иконка профиля → Настройки
  3. Раздел Доступ к API
  4. Кнопка «Создать новый токен»
  5. Название: «WB Profit» · Тип: Только чтение
  6. Раздел: Статистика (обязательно!)
  7. Создать → скопировать токен
⚠️ Не делитесь токеном с третьими лицами. Если что — его можно отозвать в том же разделе WB.
🚀
Вставьте токен и запускаем
Вставьте скопированный токен сюда — мы сохраним его и сразу загрузим ваши данные.
Токен хранится только в вашем браузере и нигде не передаётся, кроме WB.
