# SW Profit — Chrome Extension

Калькулятор прибыли продавца на Wildberries. Показывает реальную выручку с учётом СПП, комиссии, логистики (через WB Box API) и предупреждения о доступности FBS на складах — прямо на странице товара.

## Версия

0.3.9 (Manifest V3)

## Возможности

- Расчёт прибыли селлера с СПП по умолчанию
- Логистика из WB Box API (актуальные тарифы)
- Предупреждения о недоступности FBS на складах
- Селектор склада в advanced-режиме
- Расчёт точки безубыточности
- Сохранение настроек в `chrome.storage`

## Установка для разработки

1. Открой `chrome://extensions/`
2. Включи **Режим разработчика**
3. Нажми **Загрузить распакованное расширение**
4. Выбери папку `wb-extension/`
5. Расширение появится в списке. Открой любой товар на Wildberries — виджет загрузится в правом нижнем углу.

## Установка для пользователей

После публикации в Chrome Web Store — обычная установка из магазина.

## Структура

```
wb-extension/
├── manifest.json          — Manifest V3
├── background.js          — service worker
├── popup.html / popup.js  — попап при клике на иконку
├── options.html / options.js — страница настроек
├── content/
│   └── widget.js          — виджет на странице товара (40 КБ)
└── icons/                 — иконки 16/48/128 PNG
```

## Permissions

- `storage` — для сохранения пользовательских настроек

## Host permissions

- `https://*.wildberries.ru/*` — для работы виджета на страницах товаров
- `https://card.wb.ru/*` — публичный API карточки товара (тот же, что использует сайт wildberries.ru)
- `https://swprofit.ru/*` — связь с основным приложением SW Profit (будет swprofit.ru после миграции)

## Конфиденциальность

Расширение не собирает персональные данные. Все настройки хранятся локально в `chrome.storage`. Подробнее: https://swprofit.ru/privacy.html

## Связь с основным сервисом

Расширение интегрировано с SW Profit — SaaS-сервисом аналитики прибыли селлера. После регистрации в кабинете расширение получает дополнительные функции (история просмотренных товаров, сохранение сценариев расчёта).

## Бренд

SW Profit = Seller's Way Profit — путь продавца к прибыли.
